Welcome to the Banking System

The system follows ACID principles to ensure that transactions are:

  • Atomic - A transaction is either fully completed or not done at all
  • Consistent - The database remains in a consistent state before and after the transaction
  • Isolated - Transactions operate independently of each other
  • Durable - Once a transaction is committed, it remains so
